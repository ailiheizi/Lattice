# Lattice Web Client

现代化的 Lattice Web 聊天界面。

## 功能特性

### 核心功能
- ✅ 实时消息收发（WebSocket + REST API）
- ✅ 房间管理（创建、加入、离开）
- ✅ 联系人管理（添加、查看、信任级别）
- ✅ 消息搜索（全文搜索，支持房间内搜索）
- ✅ 在线状态显示
- ✅ 未读消息提醒

### 用户界面
- ✅ 现代化设计（GitHub 风格）
- ✅ 响应式布局（支持移动端）
- ✅ 侧边栏（房间列表、联系人列表、设置）
- ✅ 主聊天区域（消息列表、输入框）
- ✅ 用户信息面板
- ✅ 房间信息面板
- ✅ 设置面板

### 高级功能
- ✅ 主题切换（亮色/暗色模式）
- ✅ 表情符号选择器
- ✅ 桌面通知
- ✅ 声音提醒
- ✅ 阅读回执设置
- ✅ 消息动画效果
- ✅ 自动重连（WebSocket）
- ⏳ 文件发送（待实现）
- ⏳ 端到端加密（待实现）

## 使用方法

### 1. 启动 Store 节点

```bash
# 启动 Store 服务
cargo run --bin lattice-store

# 默认端口：
# - WebSocket: 9100
# - REST API: 9101
```

### 2. 打开 Web 界面

直接在浏览器中打开 `web/index.html` 文件，或使用 HTTP 服务器：

```bash
# 使用 Python
python -m http.server 8080 --directory web

# 使用 Node.js
npx http-server web -p 8080

# 然后访问 http://localhost:8080
```

### 3. 连接到 Store

1. 在侧边栏输入 Store API 地址（默认：`http://127.0.0.1:9101`）
2. 输入你的用户名
3. 点击 "Connect" 按钮
4. 等待连接成功（状态指示器变绿）

### 4. 开始聊天

- **创建房间**：在 Rooms 标签页底部输入房间名称，点击 "Create Room"
- **发送消息**：选择房间后，在底部输入框输入消息，按 Enter 或点击 "Send"
- **添加联系人**：切换到 Contacts 标签页，输入联系人信息，点击 "Add Contact"
- **搜索消息**：在顶部搜索框输入关键词，按 Enter 搜索

## 界面说明

### 顶部导航栏
- **Lattice** - 应用标题
- **API OK/Failed** - REST API 连接状态
- **WS Live/Off** - WebSocket 连接状态
- **☰** - 切换侧边栏显示/隐藏（移动端）
- **👤** - 打开用户信息面板
- **⚙️** - 打开设置面板

### 侧边栏
- **配置面板** - Store API 地址和用户名设置
- **Rooms** - 房间列表，显示未读消息数量
- **Contacts** - 联系人列表，显示在线状态和信任级别
- **Settings** - 应用设置（主题、通知、隐私）

### 聊天区域
- **房间标题** - 显示当前房间名称、成员数、加密状态
- **搜索栏** - 搜索消息
- **ℹ️** - 打开房间信息面板
- **消息列表** - 显示聊天消息，自动滚动到底部
- **输入区域** - 表情符号、消息输入、附件、发送按钮

## 设置选项

### 外观
- **Dark Theme** - 切换暗色/亮色主题

### 通知
- **Desktop Notifications** - 启用桌面通知（需要浏览器权限）
- **Sound Alerts** - 新消息声音提醒

### 隐私
- **Read Receipts** - 发送已读回执

## 快捷键

- **Enter** - 发送消息
- **Shift + Enter** - 换行（暂不支持）
- **Esc** - 关闭模态框

## 技术栈

- **纯 HTML/CSS/JavaScript** - 无框架依赖
- **CSS Variables** - 主题切换
- **Flexbox/Grid** - 响应式布局
- **WebSocket API** - 实时通信
- **Fetch API** - REST API 调用
- **Web Audio API** - 声音提醒
- **Notification API** - 桌面通知
- **LocalStorage** - 设置持久化

3. **消息编辑/删除** - 暂未实现
4. **@提及功能** - 暂未实现
5. **消息回复** - 暂未实现
6. **富文本编辑** - 仅支持纯文本
7. **语音/视频通话** - 暂未实现
