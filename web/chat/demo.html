Quick Start Guide

1

Start the Store Service

Open a terminal in the NextIM project directory and run:

cargo run --bin nextim-store

The Store service will start on http://127.0.0.1:9100

2

Open the Chat Interface

Click the button above or navigate to:

web/chat/index.html
Tip: For best results, serve the files using a local HTTP server:
python -m http.server 8080
3

Connect to Store

In the left sidebar, enter:

  • Store API URL: http://127.0.0.1:9100
  • Username: Your desired username (e.g., "alice")

Click Connect and wait for "API OK" and "WS Live" status.

4

Start Chatting

Create a room, invite contacts, and start messaging!

  • Create a room in the "Rooms" tab
  • Add contacts in the "Contacts" tab
  • Send messages with real-time delivery
  • Search through message history

Multi-user Testing

Test with Multiple Users

To test real-time messaging between multiple users:

  1. Start one Store service instance
  2. Open multiple browser windows or tabs
  3. Connect each window with a different username (alice, bob, charlie)
  4. Create a room in one window and note the room ID
  5. Join the same room from other windows
  6. Start chatting - messages appear instantly in all windows!
Note: For cross-machine testing, ensure the Store service is accessible on the network and update the API URL accordingly.

Keyboard Shortcuts

  • Ctrl+1 - Switch to Rooms
  • Ctrl+2 - Switch to Contacts
  • Alt+↑/↓ - Navigate rooms
Actions
  • Ctrl+K - Focus search
