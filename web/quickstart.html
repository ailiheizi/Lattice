📋 Step 1: Start the Store Node

First, make sure the Lattice Store service is running:

cargo run --bin lattice-store

The Store will start on:

  • WebSocket: ws://127.0.0.1:9100
  • REST API: http://127.0.0.1:9101

🌐 Step 2: Open the Web Client

You can access the web client in two ways:

Option A: Direct file access (recommended for development)

file:///path/to/Lattice/web/index.html

Option B: HTTP server (recommended for testing)

python -m http.server 8080 --directory web

Then open: http://localhost:8080

🔌 Step 3: Connect to Store

In the web client sidebar:

  1. Enter Store API URL: http://127.0.0.1:9101
  2. Enter your username (e.g., "alice")
  3. Click the Connect button
  4. Wait for status indicators to turn green

💬 Step 4: Start Chatting!

Once connected, you can:

  • Create a new room in the Rooms tab
  • Send messages in the chat area
  • Add contacts in the Contacts tab
  • Search messages using the search bar
  • Customize settings (theme, notifications)
