# Lattice Store 管理面板 - 功能详解

## 核心功能

### 1. 仪表板 (Dashboard)

**实时统计卡片**
- 总消息数 - 显示系统中所有消息总数
- 联系人数 - 当前联系人总数
- 房间数 - 聊天房间总数
- 存储使用 - 磁盘空间使用情况，带进度条可视化

**数据可视化图表**
- 消息趋势图 - 折线图显示消息量随时间变化
- 存储分布图 - 环形图显示已用/可用空间
- 活动统计图 - 柱状图显示系统活动情况

**自动刷新**
- 每 5 秒自动更新数据
- 实时反映系统状态变化

### 2. 消息管理 (Messages)

**消息列表**
- 卡片式展示，清晰易读
- 显示发送者、接收者、内容预览
- 消息类型标签（文本/图片/文件等）
- 时间戳显示

**搜索功能**
- 实时搜索过滤
- 支持搜索发送者、接收者、内容

**消息操作**
- 查看完整消息内容
- 删除单条消息
- 按联系人筛选消息

**自动刷新**
- 5 秒自动更新消息列表
- 保持数据最新

### 3. 联系人管理 (Contacts)

**联系人卡片**
- 头像（自动生成首字母）
- 显示名称和 ID
- 在线状态指示器（在线/离开/离线）
- 消息统计
- 最后在线时间

**状态判断**
- 在线：5 分钟内活跃
- 离开：5-30 分钟内活跃
- 离线：30 分钟以上未活跃

**联系人操作**
- 查看联系人所有消息
- 删除联系人（级联删除消息）
- 搜索联系人

**搜索功能**
- 按名称或 ID 搜索
- 实时过滤结果

### 4. 房间管理 (Rooms)

**房间信息**
- 房间图标和名称
- 房间 ID
- 成员列表（显示前 5 个，其余显示数量）
- 消息统计
- 创建时间

**房间统计**
- 成员数量
- 消息总数
- 创建时间

**房间操作**
- 查看房间消息
- 删除房间（级联删除消息）
- 搜索房间

**成员显示**
- 显示前 5 个成员
- 超过 5 个显示 "+N more"

### 5. 系统日志 (Logs)

**日志显示**
- 终端风格界面
- 彩色日志级别标识
- 时间戳
- 日志来源
- 日志消息

**日志级别**
- ERROR - 错误（红色）
- WARNING - 警告（黄色）
- INFO - 信息（蓝色）
- DEBUG - 调试（灰色）

**日志功能**
- 按级别过滤
- 搜索日志内容
- 清空日志
- 自动滚动（可选）

**自动刷新**
- 5 秒自动更新
- 可选自动滚动到底部

## 技术特性

### 模块化架构

每个功能模块独立封装：
- `Dashboard` - 仪表板模块
- `Messages` - 消息管理模块
- `Contacts` - 联系人管理模块
- `Rooms` - 房间管理模块
- `Logs` - 日志查看模块

### 生命周期管理

每个模块支持：
- `init()` - 初始化模块
- `destroy()` - 销毁模块，清理资源
- 自动刷新控制
- 事件监听器管理

### API 客户端

统一的 API 调用封装：
- RESTful API 支持
- 错误处理
- 请求/响应拦截
- 超时控制

### 图表管理

使用 Chart.js 实现：
- 折线图 - 趋势分析
- 环形图 - 占比显示
- 柱状图 - 数据对比
- 自动销毁和重建
- 响应式设计

### 状态管理

- 连接状态监控
- 页面切换管理
- 数据缓存
- 加载状态显示

### 错误处理

- API 错误捕获
- 友好错误提示
- 加载失败重试
- 空状态展示

### 用户体验

**加载状态**
- 加载动画
- 骨架屏
- 进度指示

**空状态**
- 友好的空数据提示
- 引导性文案

**通知系统**
- 成功/错误/警告通知
- 自动消失
- 动画效果

**响应式设计**
- 移动端适配
- 平板适配
- 桌面端优化

## 扩展性

### 添加新模块

1. 创建模块类
2. 实现 init/destroy 方法
3. 在 app.js 注册模块
4. 添加导航链接

### 自定义主题

修改 CSS 变量：
```css
:root {
    --primary: #3b82f6;
    --success: #10b981;
    /* ... */
}
```

### 添加新图表

使用 ChartManager：
```javascript
chartManager.createCustomChart(canvasId, config);
```

### 扩展 API

在 api.js 添加新方法：
```javascript
async getCustomData() {
    return this.get('/custom-endpoint');
}
```
